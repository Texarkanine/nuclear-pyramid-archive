# Product Context

## Target Audience

Visitors interested in Gregg Wilson's "Pyramidal Proton Model" and related articles about the Giza pyramids. The original site (nuclearpyramid.com) is dead; the audience is anyone who encounters the archived content via search engines or direct links.

## Use Cases

- Browse the original NuclearPyramid.com articles (Great Pyramid, Other Two Pyramids, Proton model) as they appeared when the site was live.
- Download the "From Gravitons to Galaxies" book referenced on the site.

## Key Benefits

- Preserves a dead website's content in a browsable, static format.
- Hosted freely and durably via GitHub Pages.

## Success Criteria

- All original content pages are browsable and visually faithful to the original site.
- Internal navigation links work correctly.
- Missing assets (images, documents) are sourced and restored.

## Key Constraints

- The site is purely static — no server-side execution.
- Original `.php` file extensions must be preserved for link correctness.
- Content is sourced from Internet Archive snapshots; some assets were not captured and must be manually supplied.
